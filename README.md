# Before You Ask for Help: A Debugging Checklist for CS Students - Landing Site

A static GitHub Pages landing page for the MVP.

## Launch steps

1. Create a public repo called `beginner-debugging-pack-site`.
2. Copy this folder into it.
3. Replace placeholder links in `index.html` and `privacy.html`:
   - `https://payhip.com/YOUR-PRODUCT-LINK`
   - `https://forms.gle/YOUR-FEEDBACK-FORM`
4. Push to GitHub.
5. Enable GitHub Pages from the repo settings.
